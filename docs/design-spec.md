# 🎨 设计规范

> 版本：v1.0 | 最后更新：2026-06-01

---

## 一、色彩系统

| 色值 | 名称 | 用途 |
|------|------|------|
| `#FFF3E0` | 背景浅橘 | 页面主背景 |
| `#FFE0B2` | 卡片橘 | 卡片、区块背景 |
| `#FF9800` | 主强调橙 | 按钮、选中状态、高亮 |
| `#F57C00` | 深强调橙 | 按钮按下态、重要文字 |
| `#FFFFFF` | 白色 | 卡片底色、文字反色 |
| `#333333` | 主文字色 | 标题、正文 |
| `#666666` | 次要文字 | 说明性文字 |
| `#999999` | 辅助文字 | 时间、难度标签 |
| `#4CAF50` | 成功绿 | 已有食材标记 |
| `#FF5252` | 缺失红 | 缺少食材标记、取消按钮 |

## 二、字体

| 用途 | 大小 | 字重 |
|------|------|------|
| 页面大标题 | 24px | Bold (700) |
| 区块标题 | 18px | SemiBold (600) |
| 正文/食材名 | 14px | Medium (500) |
| 辅助说明 | 12px | Regular (400) |
| 按钮文字 | 16px | SemiBold (600) |

## 三、圆角与阴影

| 元素 | 圆角 | 阴影 |
|------|------|------|
| 食材卡片 | 12px | `0 2px 8px rgba(0,0,0,0.08)` |
| 菜品推荐卡 | 16px | `0 4px 16px rgba(0,0,0,0.1)` |
| 按钮 | 24px（胶囊型） | `0 2px 4px rgba(0,0,0,0.12)` |
| 分类标签 | 16px（小圆角） | 无 |

---

## 四、食材卡片设计

- 尺寸：约 72×88px
- 布局：上方 emoji（36px），下方文字（14px）
- 未选中：白色背景，浅灰虚线边框
- 选中：淡橘背景（`#FFE0B2`），橙色实线边框（`#FF9800`），轻微放大（scale 1.05）
- 过渡动画：150ms ease-out

---

## 五、按钮设计

| 按钮 | 颜色 | 尺寸 |
|------|------|------|
| 主按钮（找菜/就这个） | `#FF9800` 背景 + 白色文字 | 全宽，高 52px |
| 次按钮（换一个） | 透明背景 + `#FF9800` 边框 + 橙色文字 | 全宽，高 48px |
| 小按钮（筛选确认） | 同主按钮 | 半宽，高 40px |

---

## 六、底部导航

- 4 个 Tab：首页 🏠 | 收藏 ❤️ | 历史 🕐 | 冰箱 🧊
- 选中态：橙色图标 + 文字
- 未选中态：灰色（`#999999`）
- 高度：56px（含安全区）

---

## 七、动画与微交互

| 场景 | 效果 |
|------|------|
| 食材选中 | 缩放弹跳（scale 1 → 0.9 → 1.05 → 1） |
| 找菜按钮 | 加载时显示旋转食材 emoji |
| 推荐结果出现 | 从下方滑入（slide-up），300ms |
| 换一个 | 卡片左右翻转切换，250ms |
| 收藏 | 心形跳动动画 |
| 页面切换 | 标准右滑进入/返回 |

---

## 八、空状态设计

- 收藏为空：🥘 emoji + "还没有收藏的菜品"
- 历史为空：📝 emoji + "还没有查看记录"
- 冰箱为空：🛒 emoji + "去首页添加食材吧"
- 搜索无结果：🔍 emoji + "试试减少一些筛选条件"
