# 🍳 "今天吃什么" — 需求文档

> 版本：v1.0 | 最后更新：2026-06-01

---

## 一、项目概述

一款帮助用户根据家中现有食材，智能推荐可制作菜品的跨平台手机应用（iOS + Android）。无需注册登录，打开即用。UI 风格以淡橘黄色为底色，生动活泼。

---

## 二、目标用户

- 想做饭但不想专门采购食材的人
- 厨房新手，不知道现有食材能做什么
- 想减少食物浪费、清冰箱的人

---

## 三、核心功能

### 3.1 食材选择（首页）

| 项目 | 描述 |
|------|------|
| 展示方式 | 图标网格（emoji + 中文名称） |
| 食材数量 | 约 80–100 种中餐常见食材 |
| 分类方式 | 蔬菜、肉类、豆制品/蛋类、调味料、主食 |
| 交互 | 点击选中/取消，选中状态有明显视觉反馈 |
| 已选数量 | 顶部栏显示已选食材数量 |

### 3.2 智能菜品推荐

| 项目 | 描述 |
|------|------|
| 触发方式 | 选中食材后点击「找菜」按钮 |
| 生成方式 | 调用 AI API，基于已选食材实时生成推荐 |
| 展示信息 | 菜品名称、食材匹配度、烹饪时间、难度等级 |
| 就这个 | 进入菜品详情页，查看制作步骤 |
| 换一个 | AI 生成另一个不同的菜品推荐（排除已出现的） |

### 3.3 菜品详情 / 制作步骤

- 菜品名称、封面 emoji
- 食材匹配情况（已有/缺哪些）
- 烹饪时间、难度、菜系
- 步骤化展示（步骤 1、2、3…）
- 每步简要说明
- 收藏按钮

---

## 四、辅助功能

### 4.1 我的冰箱（食材管理）
- 保存当前食材清单
- 下次打开自动加载
- 快速增删食材
- 用完的食材可一键移除

### 4.2 筛选偏好
- 菜系筛选：川菜、粤菜、鲁菜、苏菜、家常菜、其他
- 难度筛选：简单、中等、困难
- 时间筛选：≤15分钟、≤30分钟、≤1小时、不限

### 4.3 收藏功能
- 菜品详情页收藏/取消收藏
- 独立收藏页面查看所有收藏
- 本地持久化存储

### 4.4 历史记录
- 自动记录查看过的菜品
- 按时间倒序展示
- 支持清除全部历史

## 五、页面结构

```
├── 首页 Tab — 食材选择 + 推荐入口
│   ├── 分类标签栏（蔬菜 | 肉类 | 豆蛋 | 调料 | 主食）
│   ├── 食材图标网格
│   ├── 底部已选食材横条 + 数量
│   └── 「找菜」按钮
├── 推荐结果页（Stack 推入）
│   ├── 菜品信息卡片
│   ├── 食材匹配情况
│   ├── 「就这个」按钮
│   └── 「换一个」按钮
├── 菜品详情页（Stack 推入）
│   ├── 食材匹配标签
│   ├── 制作步骤列表
│   └── 收藏按钮（心形图标）
├── 收藏 Tab
│   └── 收藏菜品列表
├── 历史 Tab
│   └── 历史记录列表
└── 我的冰箱 Tab
    ├── 已存食材网格
    ├── 添加/删除食材
    └── 一键清空
```

---

## 六、非功能需求

- 离线可用：本地存储的历史、收藏、冰箱数据不依赖网络
- AI 调用需要网络连接，无网络时给出友好提示
- 响应速度：食材点击反馈 < 100ms，AI 推荐结果 < 10s
- 数据隐私：所有数据仅存本地，不上传服务器
